# GenAI Email Generator

This is a beginner Generative AI project that generates professional emails using prompt engineering and the Gemini API.

## Features
- Generates job application emails
- Allows different tones (formal, confident, polite)
- Uses user inputs for:
  - Name
  - Job Role
  - Skills

## Technologies Used
- Python
- Gemini API
- Prompt Engineering
- Large Language Models (LLMs)

## Example Input
Name: Sameena  
Role: Generative AI  
Skills: LLM, RAG

## Project Purpose
This project demonstrates basic knowledge of:
- Prompt engineering
- LLM usage
